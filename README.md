## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Customize engine specifications and contact information as needed

## 📝 Customization

### Update Engine Specifications
Edit the engine cards in `index.html`:
```html
<h3>Your Engine Name</h3>
<p class="displacement">Your Displacement</p>
<ul class="specs">
    <li><strong>Power:</strong> Your HP</li>
    <li><strong>Torque:</strong> Your Torque</li>
    <!-- etc -->
</ul>
```

### Change Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary: #ff0000;
    --secondary: #1a1a1a;
    --accent: #ffb800;
    /* etc */
}
```

### Update Contact Information
Edit the contact details in `index.html`:
```html
<p>+1 (555) 123-4567</p>
<p>info@ttfracing.com</p>
```
